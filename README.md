# Saint Jo

## Dependencies

- NodeJS
- NPM
- Shopify CLI

## Development

1. Clone the repo
```bash
$ git clone <repo>
```

2. Install dependencies
```bash
$ npm i
```

3. Start the Tailwind compiler
```bash
$ npm run tw
```

4. Start the Shopify dev server
```bash
$ shopify theme dev
```

## Production

```bash
$ shopify theme publish
```
